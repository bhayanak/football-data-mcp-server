# Changelog

All notable changes to the "Football Data MCP Server" extension will be documented in this file.

## [0.1.0] - 2026-04-01

### Added

- Initial release
- MCP server integration providing 9 football data tools
- Support for live scores, standings, teams, players, and top scorers
- Configurable API key, cache TTL, rate limiting, and timeout settings
- Commands: Show Server Health, Restart MCP Server
- Automatic configuration change detection with restart prompt
